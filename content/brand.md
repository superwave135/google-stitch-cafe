# Fig & Filter — Brand Basics

Use these values in your DESIGN.md and pass them to your AI assistant. Do not invent additional brand elements.

| Element | Value |
|---|---|
| Name | Fig & Filter |
| Tagline | Roasted here. Baked here. Poured for you. |
| Voice | Warm, plainspoken, a little dry-humoured. Never gushing, never corporate. |
| Primary colour | Deep fig purple `#4A2C4D` |
| Accent colour | Warm amber `#D9A441` |
| Background | Warm off-white `#FAF6F0` |
| Text colour | Near-black `#221E1F` |
| Headings | A serif with character (e.g. Georgia or a similar system serif) |
| Body | A clean sans-serif (e.g. system-ui stack) |

Accessibility note: fig purple on off-white and near-black on off-white both pass the Web Content Accessibility Guidelines (WCAG) AA contrast level for body text. Amber is for accents and highlights only — do not use it for body text on the off-white background.
